# Hawaii Gallery

A private gallery app for photos and videos from a Hawaii trip.

## Stack
- Next.js
- AWS-friendly architecture: Amplify, Cognito, S3, DynamoDB, Lambda
- Local demo auth and file storage for now

## Run locally

```bash
npm install
npm run dev
```

## Next steps
- Replace demo auth with Amazon Cognito
- Replace local filesystem storage with S3
- Move metadata to DynamoDB
- Add map-based browsing
